## Project Overview
This system automates the evaluation of external university course credits for transfer eligibility. It uses a Multi-Agent architecture to retrieve data, verify grades, and compare syllabi content.

## Key Features
- **Multi-Agent Orchestration**: Coordination between internal and external university agents.
- **Automated Retrieval**: Fetches student transcripts and course details via Google ADK.
- **Syllabus Comparison**: Uses NLP to determine content equivalence between courses.

## Tech Stack
- **LLM**: Gemini 2.5 Flash Lite
- **Framework**: Google Agent Development Kit (ADK)
- **Environment**: Python, Jupyter, Uvicorn (Agent-to-Agent communication)
